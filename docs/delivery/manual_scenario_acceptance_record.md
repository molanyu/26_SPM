# 手动业务流程验收结果记录表

## 1. 使用说明

本文档用于记录 `docs/delivery/manual_scenario_acceptance.md` 的实际执行结果。

填写规则：

- `状态` 固定填写：`PASS`、`FAIL`、`BLOCKED`、`N/A`
- `实际结果` 写真实观察到的页面、接口或数据库结果
- `证据` 可填写截图文件名、接口响应摘要、SQL 查询结果或邮件收件截图
- `问题编号` 可填写 `ISSUE-01`、`ISSUE-02` 等，后续统一分析
- 不确定时不要强行写 PASS，优先写 `BLOCKED` 并说明卡点
